TufteGraph

Make pretty graphs with javascript, using jQuery and the <canvas> element. It is different from other charting libraries because:

  • Configuration is by dynamic functions, allowing for really compact API (very few options)
  • Layout is done via CSS rather than code

API

$(document).ready(function () {
  jQuery.tufteBar($('#awesome-graph'), {
    data: [
      // First element is the y-value
      // Other elements are arbitary - they are not used by the lib but are passed back into callback functions
      [1.0, {label: 'Dog'}],
      [1.3, {label: 'Raccoon'}]
      // etc...
    ],

    // Bar width in arbitrary units, 1.0 means the bars will be snuggled up next to each other
    barWidth: 0.8, 

    // The label on top of the bar - can contain HTML (it goes in a DIV)
    barLabel:  function(element, index) { return element[0] + 'x' }, 

    // The label on the x-axis - can contain HTML (it goes in a DIV)
    axisLabel: function(element, index) { return element[1].label }, 

    // The color of the bar
    color:     function(element, index) { return ['#E57536', '#82293B'][index % 2] }
  });
});

Hot Tips

  • You need to be using jQuery. View source on this page if you don't quite get it.
  • It only does simple bar charts.
  • By default your graph won't be that pretty because it needs to be styled. I recommend tufte-graph.css as a good starting point.
  • Find beautiful colors for you graphs at ColourLovers
  • I have tested this in Firefox 3 and Safari 3 and nothing else. Buyer beware.
  • Read Tufte's The Visual Display of Quantitative Information, it'll blow your mind. Your graphs will never be the same again.
  • Email me the cool graphs you make, I really dig graphs.
  • If you're going to make changes to this library, please fork on github and share the love.
